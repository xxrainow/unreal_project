# Unreal_project

### 개발 담당 : 캐릭터

### 각 기능
- 카메라 설정
- 움직임 설정
- 1인칭, 3인칭 결정
- 3D 캐릭터 만들기

### 세부 단계
1. 아트 에셋 만들기
2. 스켈레탈 메시 임포트하기
3. 플레이어 컨트롤하기
4. 폰 또는 캐릭터 블루프린트 만들기
5. 폰 또는 캐릭터 선택
6. 캐릭터 클래스 셋업
7. 애니메이션 블루프린트
8. 게임모드 셋업
